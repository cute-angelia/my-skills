## 第零步：时段识别（必须优先判断）

根据当前时间或用户说明，确定分析模式：

| 时段                | 时间范围         | 分析重点                                 |
| ------------------- | ---------------- | ---------------------------------------- |
| 🟢 **盘中模式**     | 09:30–15:00      | 实时封板质量、量能趋势、当下资金方向     |
| 🟡 **收盘选股模式** | 15:00–次日 09:00 | 综合评分排名、次日买点布局（主要场景）   |
| 🔵 **隔日复盘模式** | 开盘前/盘中回顾  | 对比昨日评分与今日实际表现，更新止损计划 |

> 盘中模式：封板资金和 DDX 是实时变化的，结论只代表截图时刻，需动态跟踪。
> 收盘模式：数据固化，评分最准确，是主要分析场景。

## 第一步：大盘情绪过滤（前置必做）

**在分析个股前，先判断市场整体情绪，情绪差时个股再强也要降权。**

若用户提供了涨跌停数据，用以下公式判断：

```
情绪比 = 当日涨停数 / max(跌停数, 1)
```

| 情绪比    | 市场状态    | 操作基调                        |
| --------- | ----------- | ------------------------------- |
| ≥ 4:1     | 🔴 情绪亢奋 | 正常操作，但警惕高位风险        |
| 2:1 ~ 4:1 | 🟠 情绪偏多 | 正常操作                        |
| 1:1 ~ 2:1 | 🟡 情绪中性 | 个股评分阈值上调 2 分，谨慎跟进 |
| < 1:1     | ⚫ 情绪偏空 | 暂停操作建议，分析仅供参考      |

若用户未提供涨跌停统计，跳过此步并注明"未获取大盘情绪数据"。

## 第三步：数据获取

### 数据源 A：akshare 涨停池（日期代码触发）

```python
import akshare as ak

df = ak.stock_zt_pool_em(date='YYYYMMDD')        # 当日涨停池
df_prev = ak.stock_zt_pool_previous_em(date='YYYYMMDD')  # 昨日涨停池
df_strong = ak.stock_zt_pool_strong_em(date='YYYYMMDD')  # 强势股池
```

**akshare 可用字段：**

| 字段         | 含义               | 重要性 |
| ------------ | ------------------ | ------ |
| 涨跌幅       | 当日涨跌幅         | ★★★    |
| 连板数       | 连续涨停天数       | ★★★★★  |
| 封板资金     | 封板资金总额（元） | ★★★★★  |
| 首次封板时间 | HHMMSS 格式        | ★★★★   |
| 炸板次数     | 盘中打开涨停次数   | ★★★★   |
| 换手率       | 成交量/流通股本    | ★★★★   |
| 所属行业     | 板块分类           | ★★★★   |
| 流通市值     | 元                 | ★★★    |
| 成交额       | 元                 | ★★★    |

**筛选条件语法：**

- `20260331` → 拉取该日全部涨停股
- `20260331 连板>=2` → 仅保留连板数 ≥2
- `20260331 行业=电力` → 仅保留电力行业
- `20260331 炸板=0` → 仅保留一字板
- `20260331 换手<5 封板资金>1亿` → 多条件组合

**交易日验证：** 拉取前先确认是否为交易日，非交易日提示用户。

### 数据源 B：截图/粘贴数据

#### 图片输入处理（OCR）

当用户提供的是图片文件（.jpg/.png/.jpeg 等）而非文字数据时，**必须先使用 ocr-local 技能识别文字**，再进行后续分析。

**OCR 执行流程：**

1. 确认图片路径（用户上传的图片路径由 OpenClaw 自动提供）
2. 使用 ocr-local 技能的 OCR 脚本提取文字：
   ```bash
   node ~/.openclaw/workspace/skills/ocr-local/scripts/ocr.js /path/to/image.jpg --lang chi_sim+eng
   ```
3. 从 OCR 结果中解析股票数据（股票名称、代码、各指标数值）
4. 按常规流程继续分析

**OCR 后的数据清洗要点：**
- OCR 可能将百分比识别为小数（如 10.00% → 0.10），需还原为百分比格式
- 股票名称可能被拆分（如"皇台酒业" → "皇 台 酒 业"），需合并空格
- 数字识别可能有误差（如 0 和 O 混淆），结合上下文修正
- 如果 OCR 结果不完整或乱码严重，提示用户提供文字版本

**东方财富 DDX 说明（重要）：**

东方财富 DDX 与通达信 DDX 算法不同，数值范围通常在 -5 到 +5 之间。阈值不能通用：

| 东方财富 DDX | 含义            | 通达信 DDX | 含义            |
| ------------ | --------------- | ---------- | --------------- |
| > 1.0        | 主力强势介入 🔴 | > 2.0      | 主力强势介入 🔴 |
| 0.3~1.0      | 主力明显参与    | 1.0~2.0    | 主力有明显参与  |
| 0~0.3        | 主力参与有限    | < 1.0      | 主力参与有限    |
| < 0          | 主力净流出 ⚠️   |            |                 |

**可识别字段：**

| 字段         | 含义                     | 重要性 |
| ------------ | ------------------------ | ------ |
| 涨幅         | 当日涨跌幅               | ★★★    |
| 今日增仓占比 | 当日新开仓资金/总成交    | ★★★★★  |
| 3 日增仓占比 | 近三日增仓占比（持续性） | ★★★★★  |
| DDX          | 大单净量指数             | ★★★★★  |
| 主力净比     | 主力净流入/成交额        | ★★★★   |
| 量涨速       | 成交量放大速度           | ★★     |

### 数据源 C：混合模式

用户提供日期代码拉取 akshare 数据，同时贴出截图补充 DDX/增仓等指标。按股票代码匹配合并。

## 第四步：逐指标解读

### 4.1 封板质量指标（akshare 数据）

**首次封板时间**

- 092500 前（一字板/集合竞价涨停）：最强形态 🔴
- 092500~094500（早盘封板）：强势，市场共识高 🔴
- 094500~110000：中等偏强
- 110000~140000：中等
- 140000 后（尾盘偷袭）：偏弱，次日低开概率高 ⚠️

**封成比**（自行计算：封板资金 / 成交额）

- > 50%：封板极强 🔴
- 20%~50%：封板较强
- < 20%：封板偏弱

**炸板次数**

- 0 次：筹码锁定好
- 1 次：正常分歧转一致
- 2 次：分歧较大 ⚠️
- ≥3 次：封板不坚决，主力可能出货 ⚠️⚠️

**换手率（按股票类型区分解读）**

| 类型             | 偏弱 | 正常   | 偏强    | 警示 |
| ---------------- | ---- | ------ | ------- | ---- |
| 首板             | <3%  | 5%~15% | 15%~25% | >30% |
| 连板（2~3 板）   | >15% | 3%~8%  | <3%     | >20% |
| 连板（4 板以上） | >10% | 1%~5%  | <2%     | >12% |

> 首板高换手不一定坏（底部大换手=筹码充分换手），连板高换手才是风险信号。

**流通市值**

- < 30 亿：小盘股，游资偏好，波动大
- 30~100 亿：中盘股，资金容量适中
- > 100 亿：大盘股，需要大资金推动

### 4.2 资金指标

**DDX**

- 东方财富版：> 1.0 强势 🔴 / 0.3~1.0 参与 / < 0 净流出 ⚠️
- 通达信版：> 2.0 强势 🔴 / 1.0~2.0 参与 / < 1.0 有限

**今日增仓占比**

- > 20%：资金涌入强烈 🔴
- 10%~20%：资金参与积极
- < 10%：资金参与偏弱

**3 日增仓占比 vs 今日增仓占比**

- 今日 ≈ 3 日（差值 < 5%）：资金持续稳定入场
- 今日远高于 3 日（差值 > 10%）：可能是一日游 ⚠️
- 今日远低于 3 日：资金退潮信号 ⚠️

**主力净比**

- > 3%：主力资金占比高 🔴
- 1.5%~3%：适中
- < 1.5%：偏弱

**DDX 与主力净比一致性校验**

- 方向一致（同正同负）：信号可信度高
- 方向背离：降低评分可信度，注明"指标背离" ⚠️

## 第五步：分类评分体系

### 🅰️ 首板评分（满分 22 分）

| 维度             | 评分标准                                                                             | 分值 |
| ---------------- | ------------------------------------------------------------------------------------ | ---- |
| **封板时间**     | 09:25 前 →5 分；09:25~09:45→4 分；09:45~11:00→3 分；11:00~14:00→2 分；14:00 后 →0 分 | /5   |
| **封成比**       | >50%→4 分；30~50%→3 分；15~30%→2 分；<15%→0 分                                       | /4   |
| **炸板次数**     | 0 次 →4 分；1 次 →3 分；2 次 →1 分；≥3 次 →0 分                                      | /4   |
| **DDX**          | 东财 >1.0→4 分；0.3~1.0→3 分；0~0.3→1 分；<0→0 分（通达信阈值 ×2）                   | /4   |
| **今日增仓占比** | >20%→3 分；10~20%→2 分；<10%→1 分（无此数据时用换手率替代）                          | /3   |
| **换手率**       | 8~20%（底部充分换手）→2 分；其他适中 →1 分；>35%或<3%→0 分                           | /2   |

首板评分等级：

- 18~22 分：🔴 **强** — 重点关注，次日可积极布局
- 13~17 分：🟠 **中** — 观望为主，等待明确信号
- ≤12 分：🔵 **弱** — 回避

### 🅱️ 连板评分（满分 22 分）

| 维度               | 评分标准                                                   | 分值 |
| ------------------ | ---------------------------------------------------------- | ---- |
| **连板数**         | ≥5 板 →5 分；4 板 →4 分；3 板 →3 分；2 板 →2 分            | /5   |
| **换手率（连板）** | <3%→5 分；3~6%→4 分；6~10%→2 分；>10%→0 分                 | /5   |
| **封成比**         | >50%→4 分；30~50%→3 分；<30%→1 分                          | /4   |
| **炸板次数**       | 0 次 →4 分；1 次 →2 分；≥2 次 →0 分                        | /4   |
| **DDX**            | 东财 >1.0→2 分；0.3~1.0→1 分；其他 →0 分                   | /2   |
| **3 日增仓占比**   | >15%→2 分；8~15%→1 分；<8%→0 分（无数据跳过，满分降为 20） | /2   |

连板评分等级：

- 18~22 分：🔴 **强** — 连板龙头，重点关注
- 13~17 分：🟠 **中** — 可跟，但需控制仓位
- ≤12 分：🔵 **弱** — 回避

### 🅲️ 强势股评分（满分 18 分，无涨停故去掉封板维度）

| 维度             | 评分标准                                          | 分值 |
| ---------------- | ------------------------------------------------- | ---- |
| **涨幅**         | 8~9.9%→4 分；6~8%→3 分；3~6%→2 分；<3%→1 分       | /4   |
| **DDX**          | 东财 >1.0→4 分；0.3~1.0→3 分；0~0.3→1 分；<0→0 分 | /4   |
| **今日增仓占比** | >20%→3 分；10~20%→2 分；<10%→1 分                 | /3   |
| **3 日增仓占比** | >15%→3 分；8~15%→2 分；<8%→1 分                   | /3   |
| **主力净比**     | >3%→2 分；1.5~3%→1 分；<1.5%→0 分                 | /2   |
| **量涨速**       | >150%→2 分；50~150%→1 分；<50%→0 分               | /2   |

强势股评分等级：

- 14~18 分：🔴 **强** — 可重点关注，有涨停预期
- 10~13 分：🟠 **中** — 观察，等板块联动信号
- ≤9 分：🔵 **弱** — 回避

## 第六步：龙头识别

同一板块有多只涨停时，识别龙头股。龙头股综合评分 **+2 分**。

龙头判断条件（同时满足 3 条以上）：

1. 连板数最多
2. 封成比最高（封板最坚决）
3. 首次封板时间最早
4. 流通市值适中（30~150 亿，不是最大的）
5. DDX 全板块最高
6. 换手率适中

---

## 第七步：特别警示条件

触发以下任意条件，在该股旁标注 ⚠️：

**封板类警示：**

1. 尾盘偷袭：首封时间 > 14:00
2. 频繁炸板：炸板次数 ≥ 3
3. 封板资金过小：流通市值 > 50 亿 但封板资金 < 5000 万

**资金类警示：** 4. DDX 为负：主力净流出 5. 今强 3 日弱：今日增仓 > 15% 但 3 日增仓 < 8% 6. DDX 与主力净比背离：方向相反 7. 涨停但增仓弱：涨幅 ≥9.9% 且今日增仓 < 10%

**换手类警示：** 8. 连板高换手：连板数 ≥2 且换手率 > 12% 9. 连板无量：连板数 ≥3 且换手率 < 1%

## 第八步：次日操作参考

### 可以考虑介入的信号（满足 2 条以上）

| 信号             | 具体标准                               | 说明         |
| ---------------- | -------------------------------------- | ------------ |
| 竞价强势         | 集合竞价封单量 > 昨日成交量 5%         | 主力护盘     |
| 低开企稳         | 高开 ≤2%，回落后在昨日涨停价附近获支撑 | 洗盘确认     |
| 开盘量能持续     | 首个 5 分钟成交量 > 昨日同时段 1.5 倍  | 资金认可     |
| 板块联动         | 同板块其他股票同步走强                 | 板块行情延续 |
| 缩量回调放量突破 | 小幅回调（≤3%）后再度放量上攻          | 主力控盘洗筹 |

### 应当放弃介入的信号（触发任意 1 条即放弃）

| 信号         | 具体标准             | 说明         |
| ------------ | -------------------- | ------------ |
| 高开大幅跳空 | 开盘高开 ≥5%         | 获利盘压力大 |
| 开盘即砸盘   | 竞价后立刻大卖单压盘 | 主力出货     |
| 板块整体崩塌 | 同板块多只低开或跌停 | 逻辑崩塌     |
| 大盘急跌     | 指数开盘跌幅 >1.5%   | 系统性风险   |
| 竞价低开     | 次日竞价低开 ≥2%     | 人气散了     |

### 止损设置

- **激进止损**：跌破昨日涨停价立刻离场（约 -9%）
- **稳健止损**：跌破昨日收盘价 3% 以上离场
- **时间止损**：开盘 30 分钟内无法企稳，不管价位直接离场

## 模式二：单股深度分析（SAR + MA13 + MACD 三重共振）

用户只发一个股票代码时，自动触发。

**触发条件**：消息仅包含 6 位数字代码，或"股票名称+代码"格式。

### 数据获取

**优先使用本地计算（TA-Lib）**，网络不可用时自动回退到搜索。

#### 方式一：本地计算（推荐）

**依赖库：**
```bash
# macOS
brew install ta-lib
pip3 install --break-system-packages TA-Lib pandas numpy akshare

# 验证安装
python3 -c "import talib; print('TA-Lib:', talib.__version__)"
```

**计算脚本：**
```python
import akshare as ak
import pandas as pd
import numpy as np
import talib as ta

def get_stock_technical(code: str, days: int = 60):
    """获取股票技术指标数据"""
    try:
        # 获取历史K线
        df = ak.stock_zh_a_hist(
            symbol=code, 
            period='daily', 
            start_date=(pd.Timestamp.now() - pd.Timedelta(days=days*2)).strftime('%Y%m%d'),
            adjust='qfq'
        )
        
        # 转换数据类型
        close = df['收盘'].values.astype(np.float64)
        high = df['最高'].values.astype(np.float64)
        low = df['最低'].values.astype(np.float64)
        
        # MA13
        df['MA13'] = ta.SMA(close, timeperiod=13)
        
        # MACD
        df['DIF'], df['DEA'], df['MACD'] = ta.MACD(close, fastperiod=12, slowperiod=26, signalperiod=9)
        
        # SAR
        df['SAR'] = ta.SAR(high, low)
        
        # 判断 SAR 状态
        df['SAR状态'] = df.apply(lambda x: '多头' if x['SAR'] < x['收盘'] else '空头', axis=1)
        
        # MA13 趋势（比较最近两日）
        if len(df) >= 14:
            ma13_trend = '向上' if df['MA13'].iloc[-1] > df['MA13'].iloc[-2] else '向下' if df['MA13'].iloc[-1] < df['MA13'].iloc[-2] else '走平'
        else:
            ma13_trend = '数据不足'
        
        return {
            'latest': df.iloc[-1].to_dict(),
            'ma13_trend': ma13_trend,
            'sar_status': df['SAR状态'].iloc[-1],
            'macd_status': '金叉' if df['DIF'].iloc[-1] > df['DEA'].iloc[-1] else '死叉',
            'data': df
        }
    except Exception as e:
        return {'error': str(e)}

# 使用示例
result = get_stock_technical('002730')
if 'error' not in result:
    print(f"收盘价: {result['latest']['收盘']:.2f}")
    print(f"MA13: {result['latest']['MA13']:.2f} ({result['ma13_trend']})")
    print(f"SAR: {result['latest']['SAR']:.2f} ({result['sar_status']})")
```

#### 方式二：网络搜索（回退方案）

当本地计算失败时，使用 `web_search` 搜索以下数据（关键词：`"[股票代码] [股票名称] 行情 DDX 均线 MACD"`）：

- 近期价格、成交量
- 今日/3 日增仓占比、DDX、主力净比
- 当前收盘价 vs MA13（13 日均线）
- SAR 当前值及颜色（红=多头/绿=空头）
- MACD：DIF、DEA、柱状值

### 第一重：资金面

按上述分类评分体系打分。

### 第二重：技术面 — SAR + MA13

**SAR（抛物线止损指标）**

| 状态                      | 判断        | 说明               |
| ------------------------- | ----------- | ------------------ |
| SAR 红点（在股价下方）    | 多头趋势 ✅ | 趋势做多           |
| SAR 绿点（在股价上方）    | 空头趋势 ❌ | 趋势向下，不宜介入 |
| SAR 刚转红（1~3 日内）    | 最强信号 🔴 | 最佳介入窗口       |
| SAR 红但距股价很近（<1%） | 趋势减速 ⚠️ | 主升浪可能结束     |

**MA13（13 日均线）**

| 状态                                 | 判断              |
| ------------------------------------ | ----------------- |
| 股价 > MA13 且均线向上倾斜           | 中短期趋势健康 ✅ |
| 股价 > MA13 但均线走平               | 趋势不明朗，观望  |
| 股价回踩 MA13 后反弹（当日重新站上） | 最强买点信号 🔴🔴 |
| 股价跌破 MA13                        | 趋势破坏，不碰 ❌ |

**SAR + MA13 共振评级**

- **共振强**：SAR 红 + 股价 > MA13 + 均线向上 → 技术面全面看多
- **共振中**：SAR 红 + 股价略高于 MA13（均线走平）→ 偏多，需等待
- **共振弱/背离**：SAR 绿 或 股价跌破 MA13 → 技术面看空

### 第三重：MACD

| 状态                 | 含义                | 操作参考             |
| -------------------- | ------------------- | -------------------- |
| DIF 上穿 DEA（金叉） | 短期动能转强 ✅     | 金叉后介入，信号最纯 |
| DIF > DEA 且柱状放大 | 上涨动能持续增强 ✅ | 趋势延续             |
| DIF > DEA 但柱状缩小 | 上涨动能衰减 ⚠️     | 注意顶背离风险       |
| DIF 下穿 DEA（死叉） | 动能转弱 ❌         | 不宜介入             |
| 0 轴以上金叉         | 强势金叉 🔴         | 最优信号             |
| 0 轴以下金叉         | 弱势金叉            | 需其他指标佐证       |

**顶背离警示**：股价创新高但 MACD 柱状持续缩小 → 必须标注 ⚠️

### 三重共振结论

| 共振程度    | 条件                     | 操作倾向                       |
| ----------- | ------------------------ | ------------------------------ |
| 全面共振    | 三重均看多               | 短线信号最强，重点关注次日买点 |
| 两重共振    | 两重看多，一重中性或看空 | 信号偏正面，观望等待弱项改善   |
| 单重/零共振 | 仅一重或全部看空         | 回避                           |

## 注意事项

- **DDX 版本区分**：东方财富 DDX 阈值（>1.0）≠ 通达信（>2.0），不可混用
- **不预测涨跌**：只评价资金指标强弱和买点条件，禁止说"明天会涨"
- **操作建议用词**：只用"可关注/观望/回避"，禁止用"买入/建仓/加仓"
- **次日买点是条件不是预测**：必须说"如果出现 X 信号，可以考虑介入"
- **止损必须提**：每个有操作建议的股票都要附止损位
- **首板/连板/强势股分开排名**：不用同一评分体系混排
- **板块联动优先提醒**：同板块多只强势股时，先点明板块逻辑再分析个股
- **盘中数据注明时效**：盘中截图需注明"数据截至 XX 时"
- **技术指标只用**：SAR + MA13 + MACD，不用 KDJ、RSI 及其他指标
