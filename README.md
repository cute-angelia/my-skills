# My Skills

个人技能集合，用于 OpenClaw/QClaw。

## 结构

```
my-skills/
├── my-first-skill/       # 示例技能
│   ├── SKILL.md          # 技能说明
│   ├── scripts/          # 脚本
│   ├── references/       # 参考文档
│   └── assets/           # 资源文件
└── README.md
```

## 使用方法

将技能目录复制到 `~/.qclaw/skills/` 即可使用。

## 技能列表

- **my-first-skill** - 示例技能模板
