# A 股强势股三重共振分析技能

本技能支持两种模式：

- **批量筛选模式**：用户上传股票筛选截图或多只股票数据 → 资金面评分排名 + 操作建议
- **单股深度模式**：用户只发一个股票代码 → 资金面 + SAR/MA13 + MACD 三重共振分析

---

## 模式一：批量筛选分析

### 第一步：提取数据

从截图或文字中识别以下字段（部分字段可能缺失，跳过即可）：

| 字段         | 含义                               | 重要性 |
| ------------ | ---------------------------------- | ------ |
| 涨幅         | 当日价格涨跌幅                     | ★★★    |
| 今日增仓占比 | 当日新开仓资金占总成交比例         | ★★★★★  |
| 3 日增仓占比 | 近三日新开仓资金占比（趋势持续性） | ★★★★★  |
| DDX          | 大单净量指数，主力资金净流入强度   | ★★★★★  |
| 主力净比     | 主力资金净流入占成交额比例         | ★★★★   |
| 量涨速       | 成交量放大速度                     | ★★★    |

### 第二步：逐指标解读

**今日增仓占比**

- > 20%：资金涌入强烈 🔴
- 10%~20%：资金参与积极
- < 10%：资金参与偏弱，追高风险大

**3 日增仓占比 vs 今日增仓占比**

- 两者相近（差值 < 5%）：资金入场持续稳定，趋势可靠
- 今日远高于 3 日：可能是一日游行情，持续性存疑
- 今日远低于 3 日：资金在退潮，警惕

**DDX（大单净量指数）**

- > 2.0：主力强势介入 🔴
- 1.0~2.0：主力有明显参与
- < 1.0：主力参与有限，可能是散户拉升

**主力净比**

- > 3%：主力资金占比高，信号强
- 1.5%~3%：适中
- < 1.5%：偏弱

**量涨速**

- > 2%：量能快速放大，短线爆发力强
- 0.5%~2%：适中
- < 0.5%：量能不足，涨停可信度存疑

### 第三步：综合评分（满分 14 分）

```
今日增仓占比：> 20% → 3分；10~20% → 2分；< 10% → 1分
3日增仓占比： > 15% → 3分；8~15%  → 2分；< 8%  → 1分
DDX：         > 2.0 → 3分；1.0~2.0 → 2分；< 1.0 → 1分
主力净比：    > 3%  → 3分；1.5~3%  → 2分；< 1.5% → 1分
量涨速：      > 2%  → 2分；0.5~2%  → 1分；< 0.5% → 0分
```

评分等级：13~14 分=**强**，10~12 分=**中**，≤9 分=**弱**

### 第四步：特别警示条件（触发则标注 ⚠️）

1. **涨停但增仓弱**：涨幅 ≥9.9% 且 今日增仓占比 < 10% → 可能是游资一字板
2. **增仓今强 3 日弱**：今日增仓 > 15% 但 3 日增仓 < 8% → 一日游风险
3. **DDX 与主力净比背离**：DDX 高但主力净比低（或反之）→ 信号混乱
4. **量涨速过低**：涨停股量涨速 < 0.3% → 缩量涨停，封板不牢

### 第五步：操作建议

**用词规范：只用"可关注/观望/回避"，禁止说"买入/建仓/加仓"**

| 等级                         | 操作倾向                 | 关注重点                       |
| ---------------------------- | ------------------------ | ------------------------------ |
| 强（≥13 分，无警示）         | 次日可重点关注开盘表现   | 量能是否持续，板块联动是否支撑 |
| 中（10~12 分，或 1 条警示）  | 观望为主，等待更明确信号 | 次日开盘前 5 分钟量能和方向    |
| 弱（≤9 分，或 2 条以上警示） | 回避，不参与追涨         | —                              |

### 第六步：次日买点判断

对评分 ≥10 的股票逐一说明：

**可以考虑介入（需同时满足 2 条以上）**

| 信号               | 具体标准                               |
| ------------------ | -------------------------------------- |
| 高开低走后止跌企稳 | 高开 ≤2%，回落后在昨日涨停价附近获支撑 |
| 开盘量能放大       | 首个 5 分钟成交量 > 昨日同时段 1.5 倍  |
| 竞价封单强         | 集合竞价封单量 > 昨日成交量 5%         |
| 板块联动共振       | 同板块其他股票同步走强                 |
| 缩量回调后放量突破 | 小幅回调（≤3%）后再度放量上攻          |

**应当放弃介入（触发任意 1 条即放弃）**

| 信号            | 具体标准                     |
| --------------- | ---------------------------- |
| 高开大幅跳空    | 开盘高开 ≥5%                 |
| 开盘即大量卖出  | 集合竞价后立刻出现大卖单压盘 |
| 板块整体走弱    | 同板块多只股票低开或跌停     |
| 大盘急速下跌    | 指数开盘跌幅 > 1.5%          |
| 竞价低开超过 2% | 次日集合竞价低开 ≥2%         |

**止损位**：激进止损=跌破昨日涨停价；稳健止损=跌破昨日收盘价 3%；时间止损=开盘 30 分钟内未企稳直接离场

## 模式二：单股深度分析

用户只发一个股票代码（或"代码+名称"）时，进行三重共振分析。

**触发识别**：消息仅包含 6 位数字代码，或"股票名称+代码"格式，如"600955"、"维远股份 600955"、"帮我看看 000001"

### 数据获取

使用 `web_search` 搜索以下数据（搜索关键词格式：`"[股票代码] [股票名称] 行情 DDX 均线 MACD"`）：

需要获取的数据：

- 近期价格、成交量
- 今日/3 日增仓占比、DDX、主力净比（资金面）
- 当前收盘价 vs MA13（13 日均线）
- SAR 当前值及颜色（红=多头/绿=空头）
- MACD：DIF、DEA、柱状值，以及是否金叉/死叉

如果搜索结果数据不完整，明确告知用户哪些数据未能获取，其余已有数据照常分析。

### 三重共振评判框架

#### 第一重：资金面（同批量模式的评分逻辑）

按批量模式的 14 分评分表打分，给出等级（强/中/弱）。

#### 第二重：技术面 — SAR + MA13

**SAR（抛物线止损指标）**

| 状态                        | 判断        | 说明                     |
| --------------------------- | ----------- | ------------------------ |
| SAR 红点（在股价下方）      | 多头趋势 ✅ | 趋势做多                 |
| SAR 绿点（在股价上方）      | 空头趋势 ❌ | 趋势向下，不宜介入       |
| SAR 刚刚转红（1~3 日内）    | 最强信号 🔴 | 趋势刚切换，最佳介入窗口 |
| SAR 红点但距股价很近（<1%） | 趋势减速 ⚠️ | 主升浪可能结束，谨慎追高 |

**MA13（13 日均线）**

| 状态                                     | 判断              |
| ---------------------------------------- | ----------------- |
| 股价 > MA13 且均线向上倾斜               | 中短期趋势健康 ✅ |
| 股价 > MA13 但均线走平                   | 趋势不明朗，观望  |
| 股价回踩 MA13 后反弹（当日收盘重新站上） | 最强买点信号 🔴🔴 |
| 股价跌破 MA13                            | 趋势破坏，不碰 ❌ |

**SAR + MA13 共振评级**

- **共振强**：SAR 红 + 股价 > MA13 + 均线向上 → 技术面全面看多
- **共振中**：SAR 红 + 股价略高于 MA13（均线走平）→ 技术面偏多，需等待
- **共振弱/背离**：SAR 绿 或 股价跌破 MA13 → 技术面看空，无论资金面多强都不宜介入

**震荡市特别提示**：若近期大盘处于震荡格局（指数横盘超过 2 周），SAR 频繁红绿切换为正常现象，此时信号可信度下降，需提示用户。

#### 第三重：MACD（只用 DIF/DEA/柱，不用 KDJ/RSI）

| 状态                 | 含义                | 操作参考               |
| -------------------- | ------------------- | ---------------------- |
| DIF 上穿 DEA（金叉） | 短期动能转强 ✅     | 金叉后介入，信号最纯   |
| DIF > DEA 且柱状放大 | 上涨动能持续增强 ✅ | 趋势延续，可持有或关注 |
| DIF > DEA 但柱状缩小 | 上涨动能衰减 ⚠️     | 注意顶背离风险         |
| DIF 下穿 DEA（死叉） | 动能转弱 ❌         | 不宜介入               |
| 0 轴以上金叉         | 强势金叉 🔴         | 最优信号               |
| 0 轴以下金叉         | 弱势金叉            | 需其他指标佐证         |

**顶背离警示**：股价创新高但 MACD 柱状持续缩小 → 必须标注 ⚠️

### 三重共振结论

| 共振程度    | 条件                     | 操作倾向                       |
| ----------- | ------------------------ | ------------------------------ |
| 全面共振    | 三重均看多               | 短线信号最强，重点关注次日买点 |
| 两重共振    | 两重看多，一重中性或看空 | 信号偏正面，观望等待弱项改善   |
| 单重/零共振 | 仅一重或全部看空         | 回避                           |

### 单股输出格式

用 `visualize:show_widget` 渲染：

1. **三重共振仪表盘**：三个模块并列，每个用颜色块标注状态（绿=看多/黄=中性/红=看空）
2. **各指标详情**：资金面评分卡 + SAR/MA13 状态 + MACD 数值

文字部分结构：

```
## [股票名称]（代码）深度分析

### 资金面
[评分及各指标解读]

### 技术面：SAR + MA13
[SAR状态、MA13位置、共振评级]

### 技术面：MACD
[DIF/DEA/柱状，金叉/死叉，有无背离]

### 三重共振结论
[几重共振，操作倾向一句话]

### 次日买点参考
✅ 可以介入的条件（2~3条）
❌ 放弃介入的条件（1~2条）
🛡️ 止损位建议

## 免责声明
以上为技术指标分析，不构成任何投资建议。股市有风险，投资需谨慎。
```

## 数据缺失处理

| 缺失数据        | 处理方式                                   |
| --------------- | ------------------------------------------ |
| 3 日增仓占比    | 跳过持续性分析，注明"仅凭单日数据"         |
| DDX             | 主力分析仅依赖主力净比                     |
| 量涨速          | 跳过量能评分，总分上限降为 12 分           |
| SAR 数据未获取  | 注明"SAR 数据暂缺，技术面仅参考 MA13+MACD" |
| MACD 数据未获取 | 注明"MACD 数据暂缺"，只做 SAR+MA13 分析    |

## 注意事项

- **不预测涨跌**：只说"资金面/技术面看多/看空"，不说"明天会涨"
- **操作建议用词**：只用"可关注/观望/回避"，禁止用"买入/建仓/加仓"
- **次日买点是条件不是预测**：必须说"如果出现 X 信号，可以考虑介入"
- **止损必须提**：每个有操作建议的股票都要附止损位
- **技术指标只用**：SAR + MA13 + MACD，不用 KDJ、RSI 及其他指标
- **板块联动**：多只同板块股票时，在整体概况中点明板块效应
