# A 股强势股资金分析技能

## 适用场景

用户提供以下任意形式的股票数据时触发：

- **日期代码模式**：发送类似 `20260331` 的8位日期数字，自动通过 akshare 拉取当日涨停池数据
- **日期+筛选模式**：发送 `20260331 连板>=2` 或 `20260331 行业=电力` 等筛选条件
- **截图模式**：截图中包含涨幅、今日增仓占比、3 日增仓占比、DDX、主力净比、量涨速
- **文字粘贴模式**：文字粘贴的股票列表（含上述字段）
- **直接提问**："分析一下"、"哪只股票最强"、"资金流入最好的是哪个"

---

## 数据源

### 数据源 A：akshare 涨停池（日期代码触发）

通过 `python3` 调用 akshare 获取东方财富涨停池数据，字段丰富，无需用户手动提供。

**调用方式：**

```python
import akshare as ak

# 涨停池
df = ak.stock_zt_pool_em(date='YYYYMMDD')

# 昨日涨停池（用于分析次日表现）
df_prev = ak.stock_zt_pool_previous_em(date='YYYYMMDD')

# 强势股池
df_strong = ak.stock_zt_pool_strong_em(date='YYYYMMDD')
```

**可用字段：**

| 字段 | 含义 | 短线重要性 |
|------|------|-----------|
| 涨跌幅 | 当日涨跌幅 | ★★★ |
| 换手率 | 成交量/流通股本 | ★★★★ |
| 封板资金 | 封板资金总额（元） | ★★★★★ |
| 首次封板时间 | 最早封板时间（HHMMSS） | ★★★★ |
| 最后封板时间 | 最后一次封板时间 | ★★★ |
| 炸板次数 | 盘中打开涨停次数 | ★★★★ |
| 连板数 | 连续涨停天数 | ★★★★★ |
| 涨停统计 | 如 "1/1" 表示首次封板成功 | ★★★ |
| 所属行业 | 板块分类 | ★★★★ |
| 流通市值 | 流通市值（元） | ★★★ |
| 总市值 | 总市值（元） | ★★ |
| 成交额 | 当日成交额（元） | ★★★ |
| 最新价 | 收盘价 | ★ |

**筛选条件语法：**
- `20260331` → 拉取该日全部涨停股
- `20260331 连板>=2` → 仅保留连板数≥2的股票
- `20260331 行业=电力` → 仅保留电力行业
- `20260331 炸板=0` → 仅保留一字板（未炸板）
- `20260331 换手<5` → 仅保留换手率<5%的缩量涨停
- `20260331 封板资金>1亿` → 仅保留封板资金超1亿的
- 多条件可组合：`20260331 连板>=2 炸板=0 行业=电力`

### 数据源 A 补充：东方财富 API 直接调用

当 akshare 不可用或需要更精细的数据时，可直接调用东方财富 API。

#### API 接口

```bash
# 获取涨停/强势股资金数据（批量）
curl -s "https://push2.eastmoney.com/api/qt/clist/get?pn=1&pz=50&po=1&np=1&ut=bd1d9ddb04089700cf9c27f6f7426281&fltt=2&invt=2&fid=f3&fs=m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23&fields=f12,f14,f2,f3,f62,f184,f66,f69,f72,f78,f84,f85,f92"

# 获取单只股票资金数据
curl -s "https://push2.eastmoney.com/api/qt/stock/get?secid=0.000001&fields=f57,f58,f43,f169,f170,f46,f44,f51,f52,f62,f184,f66,f69,f72,f78,f84,f85,f92"

# 获取历史资金数据（计算3日增仓）
curl -s "https://push2his.eastmoney.com/api/qt/stock/fflow/daykline/get?secid=0.000001&lmt=3&fields1=f1,f2,f3,f7&fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61,f62,f63"
```

#### 代码转换规则（secid）

股票代码需要加上市场前缀才能调用 API：

| 代码特征 | 市场 | secid 格式 | 示例 |
|----------|------|-----------|------|
| 6 开头（非688） | 沪市 | `1.代码` | 600519 → `1.600519` |
| 688 开头 | 科创板 | `1.代码` | 688001 → `1.688001` |
| 0 开头 | 深市 | `0.代码` | 000001 → `0.000001` |
| 3 开头 | 创业板 | `0.代码` | 300750 → `0.300750` |

#### API 字段映射

| API 字段 | 含义 | 对应分析字段 |
|---------|------|------------|
| f2 | 最新价 | - |
| f3 | 涨跌幅(%) | 涨幅 |
| f12 | 股票代码 | - |
| f14 | 股票名称 | - |
| f62 | 主力净流入(元) | 主力净流入额 |
| f66 | 超大单净流入(元) | - |
| f69 | 超大单净占比(%) | 今日增仓占比（近似） |
| f72 | 大单净流入(元) | - |
| f78 | 中单净流入(元) | - |
| f84 | 小单净流入(元) | - |
| f85 | 小单净占比(%) | - |
| f92 | 量比 | 量涨速（近似） |
| f184 | 主力净占比(%) | 主力净比 |

#### 数据转换规则

- **涨幅** = f3（直接使用）
- **主力净比** = f184（直接使用，百分比）
- **今日增仓占比** = f69（超大单净占比，作为主力增仓的近似值）
- **量涨速** = f92 × 100（量比转换为百分比，如 2.5 → 250%）
- **DDX**：API 不直接提供，使用主力净比(f184)作为替代指标
- **3日增仓占比**：调用历史资金数据接口，取近3日 f69 累计计算

### 数据源 B：通达信/同花顺截图或粘贴数据

用户提供包含 DDX、增仓占比、主力净比等专有指标的截图或文字。

**可用字段：**

| 字段 | 含义 | 重要性 |
|------|------|--------|
| 涨幅 | 当日价格涨跌幅 | ★★★ |
| 今日增仓占比 | 当日新开仓资金占总成交比例 | ★★★★★ |
| 3 日增仓占比 | 近三日新开仓资金占比（趋势持续性） | ★★★★★ |
| DDX | 大单净量指数，主力资金净流入强度 | ★★★★★ |
| 主力净比 | 主力资金净流入占成交额比例 | ★★★★ |
| 量涨速 | 成交量放大速度 | ★ |

### 数据源 C：混合模式

用户提供日期代码拉取 akshare 数据，同时贴出通达信截图补充 DDX/增仓等指标。两套数据按股票代码匹配合并分析。

## 分析流程

### 第一步：获取数据

**如果是日期代码模式：**

1. 解析日期和筛选条件
2. 用 python3 调用 akshare 拉取涨停池数据
3. 如有筛选条件，过滤数据
4. 同时拉取昨日涨停池，对比次日表现（可选）

**如果是截图/粘贴模式：**

1. 从截图或文字中提取字段
2. 识别股票代码/名称

**如果是混合模式：**

1. 分别获取两套数据
2. 按股票代码匹配合并

### 第二步：逐指标解读

#### akshare 专属指标解读

**连板数**
- ≥3 连板：高度关注 🔴，情绪龙头候选
- 2 连板：重点关注，确认板块逻辑
- 1 板（首板）：需其他指标确认强度

**封板资金**
- > 5亿：封板坚决，主力态度明确 🔴
- 1~5亿：封板稳健
- < 1亿：封板力度偏弱，需结合市值判断

**封成比**（封板资金 / 成交额，自行计算）
- > 50%：封板极强 🔴
- 20%~50%：封板较强
- < 20%：封板偏弱

**首次封板时间**
- 093000（一字板）：最强形态，但需看封板资金是否够大
- 093000~094500（早盘封板）：强势，市场共识度高 🔴
- 094500~110000（上午封板）：中等偏强
- 110000~140000（午盘封板）：中等
- 140000~150000（尾盘封板）：偏弱，偷袭嫌疑 ⚠️

**炸板次数**
- 0次（一字板/T字板）：筹码锁定好
- 1次：正常分歧转一致
- ≥2次：分歧大，封板不坚决 ⚠️

**换手率**
- < 3%：缩量涨停，筹码锁定好（适合连板股）
- 3%~8%：正常换手
- 8%~15%：放量涨停，分歧较大
- > 15%：高换手，获利盘出逃严重 ⚠️
- 注意：首板高换手不一定是坏事（可能是底部换手），连板高换手则需警惕

**流通市值**
- < 30亿：小盘股，游资偏好，波动大
- 30~100亿：中盘股，资金容量适中
- > 100亿：大盘股，需要大资金推动

#### 通达信专有指标解读

**今日增仓占比**
- > 20%：资金涌入强烈 🔴
- 10%~20%：资金参与积极
- < 10%：资金参与偏弱

**3 日增仓占比 vs 今日增仓占比**
- 两者相近（差值 < 5%）：资金入场持续稳定
- 今日远高于 3 日：可能是一日游行情
- 今日远低于 3 日：资金在退潮

**DDX（大单净量指数）**
- > 2.0：主力强势介入 🔴
- 1.0~2.0：主力有明显参与
- < 1.0：主力参与有限

**主力净比**
- > 3%：主力资金占比高 🔴
- 1.5%~3%：适中
- < 1.5%：偏弱

### 第三步：综合评分

#### akshare 数据评分（满分 20 分）

```
连板数：     ≥3板 → 4分；2板 → 3分；1板 → 1分
封成比：     >50% → 4分；20~50% → 3分；<20% → 1分
首次封板时间：09:30前(一字板) → 4分；09:30~09:45 → 3分；09:45~14:00 → 2分；14:00后 → 0分
炸板次数：   0次 → 4分；1次 → 3分；≥2次 → 1分
换手率：     <3%(连板)或3~8%(首板) → 4分；其他适中 → 2分；>15%或异常 → 0分
```

评分等级：
- 17~20 分：**强** 🔴 — 资金面全面共振
- 13~16 分：**中** 🟠 — 资金面偏正面，可关注
- ≤12 分：**弱** 🔵 — 资金面疲弱，谨慎对待

#### 通达信数据评分（满分 14 分，保持原规则）

```
今日增仓占比：> 20% → 3分；10~20% → 2分；< 10% → 1分
3日增仓占比：> 15% → 3分；8~15% → 2分；< 8% → 1分
DDX：        > 2.0 → 3分；1.0~2.0 → 2分；< 1.0 → 1分
主力净比：   > 3%  → 3分；1.5~3% → 2分；< 1.5% → 1分
量涨速：     > 2%  → 2分；0.5~2% → 1分；< 0.5% → 0分
```

#### 混合评分

当两套数据都有时，取各维度最高分项合并，权重：
- 连板数 + 封板质量：40%
- 资金流向（DDX/增仓）：35%
- 换手率 + 量能：15%
- 板块联动：10%

### 第四步：特别警示条件

**akshare 专属警示：**
1. **尾盘偷袭板**：首次封板时间 > 14:00 → 次日低开概率高 ⚠️
2. **频繁炸板**：炸板次数 ≥ 3 → 封板不坚决，主力可能在出货 ⚠️
3. **高换手连板**：连板 ≥ 2 且换手率 > 15% → 获利盘出逃严重 ⚠️
4. **小封板资金大市值**：流通市值 > 50亿但封板资金 < 5000万 → 主力投入不足 ⚠️
5. **首板无行业联动**：连板 = 1 且同行业无其他涨停 → 孤军深入 ⚠️

**通达信专属警示：**
6. **涨停但增仓弱**：涨幅 ≥9.9% 且今日增仓占比 < 10% ⚠️
7. **增仓今强 3 日弱**：今日增仓 > 15% 但 3 日增仓 < 8% ⚠️
8. **DDX 与主力净比背离**：DDX 高但主力净比低 ⚠️

### 第五步：操作建议

**强势股（评分 ≥17(akshare) 或 ≥13(通达信)，无警示）**
- 操作倾向：次日可重点关注开盘表现
- 关注重点：开盘后量能是否持续，是否有板块联动支撑
- 风险提示：已涨停次日存在高开低走风险，需设好止损位

**中等股（评分 13~16(akshare) 或 10~12(通达信)，或有 1 条警示）**
- 操作倾向：观望为主，等待更明确信号
- 关注重点：次日开盘前 5 分钟量能和方向
- 风险提示：资金面信号不够纯粹，追高性价比低

**弱势股（评分 ≤12(akshare) 或 ≤9(通达信)，或有 2 条以上警示）**
- 操作倾向：回避，不参与追涨

### 第六步：次日买点判断

对评分达标的股票，逐一说明次日什么情况可以考虑介入，什么情况应该放弃。

#### 可以考虑介入的信号（需同时满足 2 条以上）

| 信号               | 具体标准                                        | 说明           |
| ------------------ | ----------------------------------------------- | -------------- |
| 高开低走后止跌企稳 | 开盘高开 2%以内，回落后在昨日涨停价附近获得支撑 | 洗盘后二次确认 |
| 开盘量能持续放大   | 开盘首个 5 分钟成交量 > 昨日同时段 1.5 倍       | 资金继续认可   |
| 竞价封单强         | 集合竞价阶段封单量 > 昨日成交量 5%              | 主力护盘意愿强 |
| 板块联动共振       | 同板块其他股票同步走强                          | 板块行情延续   |
| 缩量回调后放量突破 | 小幅回调（≤3%）后再度放量上攻                   | 主力控盘洗筹   |

#### 应当放弃介入的信号（触发任意 1 条即放弃）

| 信号            | 具体标准                     | 说明                       |
| --------------- | ---------------------------- | -------------------------- |
| 高开大幅跳空    | 开盘高开 ≥5%                 | 获利盘压力大，容易冲高回落 |
| 开盘即大量卖出  | 集合竞价后立刻出现大卖单压盘 | 主力出货信号               |
| 板块整体走弱    | 同板块多只股票低开或跌停     | 板块逻辑崩塌               |
| 大盘急速下跌    | 指数开盘跌幅>1.5%            | 系统性风险压制             |
| 竞价低开超过 2% | 次日集合竞价低开 ≥2%         | 持仓者信心不足，人气散了   |

#### 止损位设置建议

- **激进止损**：跌破昨日涨停价（即跌幅超过约-9%）立刻离场
- **稳健止损**：跌破昨日收盘价 3%以上离场
- **时间止损**：开盘 30 分钟内无法企稳反弹，不管价位直接离场

## 数据缺失处理

| 缺失字段     | 处理方式                                   |
| ------------ | ------------------------------------------ |
| 3 日增仓占比 | 跳过持续性分析，注明"仅凭单日数据"         |
| DDX          | 主力分析仅依赖主力净比，降低该维度权重     |
| 量涨速       | 跳过量能评分项，总分上限降为 12 分         |
| 主力净比     | 主力分析仅依赖 DDX                         |
| 连板数       | 默认为首板，连板维度得1分                   |
| 封板资金     | 跳过封成比分析，仅用时间+炸板判断封板质量  |

## 注意事项

- **不预测涨跌**：只评价资金指标的强弱和买点条件，不说"明天会涨"
- **操作建议用词**：只用"可关注/观望/回避"，禁止用"买入/建仓/加仓"
- **次日买点是条件，不是预测**：必须说"如果出现 X 信号，可以考虑介入"
- **板块联动**：如多只股票同属一个板块，在整体概况中点明
- **涨停类型区分**：一字板 vs T 字板的分析逻辑不同
- **止损必须提**：每个有操作建议的股票，都要附上止损位
- **日期验证**：拉取数据前先确认日期是否为交易日，如非交易日提示用户
