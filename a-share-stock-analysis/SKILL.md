# A Share Stock Analysis

## 分析模式

### 模式一：数据输入（截图/文字）

用户上传截图或粘贴包含以下字段的股票数据，直接进入分析流程。

### 模式二：股票代码查询

用户输入股票代码或名称，从东方财富获取实时资金数据后进行分析。

**触发示例：**
- "分析一下 000001"
- "看看平安银行"
- "帮我分析 600519, 000858"
- "茅台的资金情况怎么样"

---

## 数据获取（模式二专用）

当用户输入股票代码时，调用东方财富 API 获取以下数据：

### API 接口

```bash
# 获取涨停/强势股资金数据（批量）
curl -s "https://push2.eastmoney.com/api/qt/clist/get?pn=1&pz=50&po=1&np=1&ut=bd1d9ddb04089700cf9c27f6f7426281&fltt=2&invt=2&fid=f3&fs=m:0+t:6,m:0+t:80,m:1+t:2,m:1+t:23&fields=f12,f14,f2,f3,f62,f184,f66,f69,f72,f78,f84,f85,f92"

# 获取单只股票资金数据
curl -s "https://push2.eastmoney.com/api/qt/stock/get?secid=0.000001&fields=f57,f58,f43,f169,f170,f46,f44,f51,f52,f62,f184,f66,f69,f72,f78,f84,f85,f92"
```

### 字段映射

| API 字段 | 含义 | 对应分析字段 |
|---------|------|------------|
| f12 | 股票代码 | - |
| f14 | 股票名称 | - |
| f2 | 最新价 | - |
| f3 | 涨跌幅(%) | 涨幅 |
| f62 | 主力净流入(元) | 主力净流入额 |
| f184 | 主力净占比(%) | 主力净比 |
| f66 | 超大单净流入(元) | - |
| f69 | 超大单净占比(%) | 今日增仓占比（近似） |
| f72 | 大单净流入(元) | - |
| f78 | 中单净流入(元) | - |
| f84 | 小单净流入(元) | - |
| f85 | 小单净占比(%) | - |
| f92 | 量比 | 量涨速（近似） |

### 数据转换规则

东方财富 API 返回的数据需要进行以下转换才能用于分析：

1. **涨幅** = f3（直接使用）
2. **主力净比** = f184（直接使用，百分比）
3. **今日增仓占比** = f69（超大单净占比，作为主力增仓的近似值）
4. **量涨速** = f92 * 100（量比转换为百分比，如 2.5 → 250%）
5. **DDX**：需要额外计算或使用第三方数据，暂用主力净比(f184)近似
6. **3日增仓占比**：需要调用历史数据接口获取

**注意：** 东方财富 API 不直接提供 DDX 和 3 日增仓占比，需要：
- DDX：使用主力净比(f184)作为替代指标
- 3日增仓占比：调用多日数据累计计算

```bash
# 获取历史资金数据（计算3日增仓）
curl -s "https://push2his.eastmoney.com/api/qt/stock/fflow/daykline/get?secid=0.000001&lmt=3&fields1=f1,f2,f3,f7&fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61,f62,f63"
```

### 代码转换

股票代码需要加上市场前缀：
- 深市股票：`secid=0.代码`（如 000001 → 0.000001）
- 沪市股票：`secid=1.代码`（如 600519 → 1.600519）
- 创业板：`secid=0.代码`（3 开头）
- 科创板：`secid=1.代码`（688 开头）

**判断规则：**
- 代码以 6 开头 → 沪市(secid=1)
- 代码以 0/3 开头 → 深市(secid=0)
- 代码以 688 开头 → 科创板(secid=1)

## 分析流程

### 第一步：提取数据

从截图或文字中识别以下字段（部分字段可能缺失，跳过即可）：

| 字段         | 含义                               | 重要性 |
| ------------ | ---------------------------------- | ------ |
| 涨幅         | 当日价格涨跌幅                     | ★★★    |
| 今日增仓占比 | 当日新开仓资金占总成交比例         | ★★★★★  |
| 3 日增仓占比 | 近三日新开仓资金占比（趋势持续性） | ★★★★★  |
| DDX          | 大单净量指数，主力资金净流入强度   | ★★★★★  |
| 主力净比     | 主力资金净流入占成交额比例         | ★★★★   |
| 量涨速       | 成交量放大速度                     | ★      |

### 第二步：逐指标解读

对每只股票按以下标准逐一评价：

**今日增仓占比**

- > 20%：资金涌入强烈 🔴
- 10%~20%：资金参与积极
- < 10%：资金参与偏弱，追高风险大

**3 日增仓占比 vs 今日增仓占比**

- 两者相近（差值 < 5%）：资金入场持续稳定，趋势可靠
- 今日远高于 3 日：可能是一日游行情，持续性存疑
- 今日远低于 3 日：资金在退潮，警惕

**DDX（大单净量指数）**

- > 2.0：主力强势介入 🔴
- 1.0~2.0：主力有明显参与
- < 1.0：主力参与有限，可能是散户拉升

**主力净比**

- > 3%：主力资金占比高，信号强
- 1.5%~3%：适中
- < 1.5%：偏弱

**量涨速**

- > 2%：量能快速放大，短线爆发力强
- 0.5%~2%：适中
- < 0.5%：量能不足，涨停可信度存疑（尤其对于 10%涨停股）

### 第三步：综合评分

对每只股票打分（满分 14 分）：

```
今日增仓占比：> 20% → 3分；10~20% → 2分；< 10% → 1分
3日增仓占比：> 15% → 3分；8~15% → 2分；< 8% → 1分
DDX：        > 2.0 → 3分；1.0~2.0 → 2分；< 1.0 → 1分
主力净比：   > 3%  → 3分；1.5~3% → 2分；< 1.5% → 1分
量涨速：     > 2%  → 2分；0.5~2% → 1分；< 0.5% → 0分
```

评分等级：

- 13~14 分：**强** — 资金面全面共振，短线强势概率高
- 10~12 分：**中** — 资金面偏正面，可关注
- ≤9 分：**弱** — 资金面疲弱，谨慎对待

### 第四步：特别警示条件

以下情况必须在分析中单独标注 ⚠️：

1. **涨停但增仓弱**：涨幅 ≥9.9% 且 今日增仓占比 < 10% → 可能是游资一字板，缺乏持续性
2. **增仓今强 3 日弱**：今日增仓 > 15% 但 3 日增仓 < 8% → 一日游风险
3. **DDX 与主力净比背离**：DDX 高但主力净比低（或反之）→ 信号混乱，需谨慎
4. **量涨速过低**：涨停股量涨速 < 0.3% → 缩量涨停，封板不牢

### 第五步：操作建议

根据综合评分 + 警示条件，对每只股票给出分级操作建议。**用词必须保守，只说"可关注/观望/回避"，不说"买入/建仓"。**

**强势股（评分 ≥13，无警示）**

- 操作倾向：次日可重点关注开盘表现
- 关注重点：开盘后量能是否持续，是否有板块联动支撑
- 风险提示：已涨停次日存在高开低走风险，需设好止损位

**中等股（评分 10~12，或有 1 条警示）**

- 操作倾向：观望为主，等待更明确信号
- 关注重点：次日开盘前 5 分钟量能和方向
- 风险提示：资金面信号不够纯粹，追高性价比低

**弱势股（评分 ≤9，或有 2 条以上警示）**

- 操作倾向：回避，不参与追涨
- 原因说明：资金面未形成共振，涨停可信度低

### 第六步：次日买点判断

对评分 ≥10 的股票，逐一说明次日什么情况可以考虑介入，什么情况应该放弃。

#### 可以考虑介入的信号（需同时满足 2 条以上）

| 信号               | 具体标准                                        | 说明           |
| ------------------ | ----------------------------------------------- | -------------- |
| 高开低走后止跌企稳 | 开盘高开 2%以内，回落后在昨日涨停价附近获得支撑 | 洗盘后二次确认 |
| 开盘量能持续放大   | 开盘首个 5 分钟成交量 > 昨日同时段 1.5 倍       | 资金继续认可   |
| 竞价封单强         | 集合竞价阶段封单量 > 昨日成交量 5%              | 主力护盘意愿强 |
| 板块联动共振       | 同板块其他股票同步走强                          | 板块行情延续   |
| 缩量回调后放量突破 | 小幅回调（≤3%）后再度放量上攻                   | 主力控盘洗筹   |

#### 应当放弃介入的信号（触发任意 1 条即放弃）

| 信号            | 具体标准                     | 说明                       |
| --------------- | ---------------------------- | -------------------------- |
| 高开大幅跳空    | 开盘高开 ≥5%                 | 获利盘压力大，容易冲高回落 |
| 开盘即大量卖出  | 集合竞价后立刻出现大卖单压盘 | 主力出货信号               |
| 板块整体走弱    | 同板块多只股票低开或跌停     | 板块逻辑崩塌               |
| 大盘急速下跌    | 指数开盘跌幅>1.5%            | 系统性风险压制             |
| 竞价低开超过 2% | 次日集合竞价低开 ≥2%         | 持仓者信心不足，人气散了   |

#### 止损位设置建议

- **激进止损**：跌破昨日涨停价（即跌幅超过约-9%）立刻离场
- **稳健止损**：跌破昨日收盘价 3%以上离场
- **时间止损**：开盘 30 分钟内无法企稳反弹，不管价位直接离场

> 说明：止损位不是精确数字，是离场的触发条件，用户需根据自己的仓位和风险承受力调整。

## 输出格式

### 必须包含的部分

1. **数据汇总表格**：Markdown 表格，包含原始数据 + 综合评分列（用 🟢/🟡/🔴 标识强/中/弱）

2. **文字解读**：结构如下：

```markdown
## 整体概况
一句话概括这批股票的共同特征（板块、市场情绪等）

## 各指标解读
分增仓、DDX、主力净比、量涨速四个维度，点评所有股票中的亮点与弱点

## 综合排名
用表格列出排名，格式：
| 排名 | 股票名称 | 核心亮点 | 操作倾向 |

## 次日买点参考
对评分≥10的股票逐一列出：
- ✅ 可以介入的条件（列2~3条最关键的）
- ❌ 放弃介入的条件（列1~2条最关键的）
- 🛡️ 止损位建议

## ⚠️ 特别警示
（如有触发警示条件的股票，在此单独说明）

## 免责声明
固定结尾：以上为技术指标分析，不构成任何投资建议。股市有风险，投资需谨慎。
```

## 数据缺失处理

| 缺失字段     | 处理方式                                   |
| ------------ | ------------------------------------------ |
| 3 日增仓占比 | 跳过持续性分析，在结论中注明"仅凭单日数据" |
| DDX          | 主力分析仅依赖主力净比，降低该维度权重     |
| 量涨速       | 跳过量能评分项，总分上限降为 12 分         |
| 主力净比     | 主力分析仅依赖 DDX                         |

如果字段普遍缺失（仅有涨幅），告知用户数据不足，建议补充更多指标后再分析。

## 注意事项

- **不预测涨跌**：只评价资金指标的强弱和买点条件，不说"明天会涨"之类的话
- **操作建议用词**：只用"可关注/观望/回避"，禁止用"买入/建仓/加仓"
- **次日买点是条件，不是预测**：必须说"如果出现 X 信号，可以考虑介入"，不说"明天可以买"
- **板块联动**：如多只股票同属一个板块，在整体概况中点明，因为板块效应会影响个股可信度
- **涨停类型区分**：一字板（开盘即涨停）与 T 字板（盘中打开过）的增仓数据含义不同，如能从量涨速判断，应注明
- **止损必须提**：每个有操作建议的股票，都要附上止损位，不能只说买点不说风险
