# A 股强势股资金分析技能（东方财富 DDX 版）

## 适用场景与触发方式

| 触发方式      | 示例                                     |
| ------------- | ---------------------------------------- |
| **日期代码**  | `20260331` — 自动拉取当日涨停池          |
| **日期+筛选** | `20260331 连板>=2 炸板=0`                |
| **截图/粘贴** | 含涨幅、DDX、增仓占比、主力净比等字段    |
| **直接提问**  | "分析一下"、"哪只最强"、"资金流入最好的" |

---

## 第零步：时段识别（必须优先判断）

**根据当前时间或用户说明，确定分析模式，不同模式侧重点不同。**

| 时段                | 时间范围         | 分析重点                                 |
| ------------------- | ---------------- | ---------------------------------------- |
| 🟢 **盘中模式**     | 09:30–15:00      | 实时封板质量、量能趋势、当下资金方向     |
| 🟡 **收盘选股模式** | 15:00–次日 09:00 | 综合评分排名、次日买点布局               |
| 🔵 **隔日复盘模式** | 开盘前/盘中回顾  | 对比昨日评分与今日实际表现，更新止损计划 |

> 盘中模式：封板资金和 DDX 数据是实时变化的，结论只代表截图时刻，需动态跟踪。
> 收盘模式：数据固化，评分最准确，是主要分析场景。

## 第一步：大盘情绪过滤（前置必做）

**在分析个股前，先判断市场整体情绪，情绪差时个股再强也要降权。**

### 情绪判断方法

若用户提供了涨跌停数据，用以下公式判断：

```
情绪比 = 当日涨停数 / max(跌停数, 1)
```

| 情绪比    | 市场状态    | 操作基调                        |
| --------- | ----------- | ------------------------------- |
| ≥ 4:1     | 🔴 情绪亢奋 | 正常操作，但警惕高位风险        |
| 2:1 ~ 4:1 | 🟠 情绪偏多 | 正常操作                        |
| 1:1 ~ 2:1 | 🟡 情绪中性 | 个股评分阈值上调 2 分，谨慎跟进 |
| < 1:1     | ⚫ 情绪偏空 | 暂停操作建议，分析仅供参考      |

若用户未提供涨跌停统计，跳过此步并注明"未获取大盘情绪数据，以下分析仅基于个股指标"。

## 第三步：数据获取

### 数据源 A：akshare 涨停池（日期代码触发）

```python
import akshare as ak

df = ak.stock_zt_pool_em(date='YYYYMMDD')        # 当日涨停池
df_prev = ak.stock_zt_pool_previous_em(date='YYYYMMDD')  # 昨日涨停池（连板参考）
df_strong = ak.stock_zt_pool_strong_em(date='YYYYMMDD')  # 强势股池
```

**筛选条件语法：**

- `20260331 连板>=2` → 仅保留连板数 ≥2
- `20260331 炸板=0` → 仅保留一字板
- `20260331 行业=电力` → 仅保留电力行业
- `20260331 换手<5 封板资金>1亿` → 多条件组合

**交易日验证：** 拉取前先确认是否为交易日，非交易日提示用户。

### 数据源 B：截图/粘贴（东方财富 DDX 版）

**东方财富 DDX 说明（重要）：**

东方财富的 DDX 与通达信 DDX 算法不同，是基于**大单净量/流通股本**的改良指标，数值范围通常在 -5 到 +5 之间（通达信版范围更广）。阈值不能通用，需使用下表：

| 东方财富 DDX | 含义            | 重要性 |
| ------------ | --------------- | ------ |
| > 1.0        | 主力强势介入 🔴 | ★★★★★  |
| 0.3 ~ 1.0    | 主力有明显参与  | ★★★★   |
| 0 ~ 0.3      | 主力参与有限    | ★★★    |
| < 0          | 主力净流出 ⚠️   | —      |

**可识别字段（截图或文字均可）：**

| 字段              | 含义                     | 重要性 |
| ----------------- | ------------------------ | ------ |
| 涨幅              | 当日涨跌幅               | ★★★    |
| 今日增仓占比      | 当日新开仓资金/总成交    | ★★★★★  |
| 3 日增仓占比      | 近三日增仓占比（持续性） | ★★★★★  |
| DDX（东方财富版） | 大单净量指数             | ★★★★★  |
| 主力净比          | 主力净流入/成交额        | ★★★★   |
| 量涨速            | 成交量放大速度           | ★★     |

**akshare 特有字段：**

| 字段         | 含义             | 重要性 |
| ------------ | ---------------- | ------ |
| 连板数       | 连续涨停天数     | ★★★★★  |
| 封板资金     | 封板资金总额     | ★★★★★  |
| 首次封板时间 | HHMMSS 格式      | ★★★★   |
| 炸板次数     | 盘中打开涨停次数 | ★★★★   |
| 换手率       | 成交量/流通股本  | ★★★★   |
| 所属行业     | 板块分类         | ★★★★   |
| 流通市值     | 元               | ★★★    |

## 第四步：逐指标解读

### 4.1 封板质量指标（akshare 数据）

**首次封板时间**

- 092500 前（一字板）：最强形态 🔴
- 092500~094500（早盘封板）：强势，市场共识高 🔴
- 094500~110000：中等偏强
- 110000~140000：中等
- 140000 后（尾盘偷袭）：偏弱，次日低开概率高 ⚠️

**封成比**（自行计算：封板资金 / 成交额）

- > 50%：封板极强 🔴
- 20%~50%：封板较强
- < 20%：封板偏弱

**炸板次数**

- 0 次：筹码锁定好
- 1 次：正常分歧转一致
- 2 次：分歧较大 ⚠️
- ≥3 次：封板不坚决，主力可能出货 ⚠️⚠️

**换手率（按股票类型区分解读）**

| 类型             | 偏弱 | 正常   | 偏强    | 警示 |
| ---------------- | ---- | ------ | ------- | ---- |
| 首板             | <3%  | 5%~15% | 15%~25% | >30% |
| 连板（2~3 板）   | >15% | 3%~8%  | <3%     | >20% |
| 连板（4 板以上） | >10% | 1%~5%  | <2%     | >12% |

> 首板高换手不一定坏：底部大换手 = 筹码充分换手，是连板的基础；连板高换手 = 获利盘出逃，是风险信号。

### 4.2 资金指标（东方财富 DDX 版）

**DDX（东方财富版）**

- > 1.0：主力强势介入 🔴
- 0.3~1.0：主力明显参与
- 0~0.3：主力参与有限
- < 0：主力净流出 ⚠️

**今日增仓占比**

- > 20%：资金涌入强烈 🔴
- 10%~20%：资金参与积极
- < 10%：资金参与偏弱

**3 日增仓占比 vs 今日增仓占比对比**

- 今日 ≈ 3 日（差值 < 5%）：资金持续稳定入场
- 今日远高于 3 日（差值 > 10%）：可能是一日游，谨慎 ⚠️
- 今日远低于 3 日：资金退潮信号 ⚠️

**主力净比**

- > 3%：主力资金占比高 🔴
- 1.5%~3%：适中
- < 1.5%：偏弱

**DDX 与主力净比一致性校验**

- 两者方向一致（同正同负）：信号可信度高
- 方向背离：降低该股评分可信度，注明"指标背离，需谨慎" ⚠️

## 第五步：分类评分体系

### 🅰️ 首板评分（满分 22 分）

| 维度              | 评分标准                                                                                     | 分值 |
| ----------------- | -------------------------------------------------------------------------------------------- | ---- |
| **封板时间**      | 09:25 前 → 5 分；09:25~09:45 → 4 分；09:45~11:00 → 3 分；11:00~14:00 → 2 分；14:00 后 → 0 分 | /5   |
| **封成比**        | >50% → 4 分；30~50% → 3 分；15~30% → 2 分；<15% → 0 分                                       | /4   |
| **炸板次数**      | 0 次 → 4 分；1 次 → 3 分；2 次 → 1 分；≥3 次 → 0 分                                          | /4   |
| **DDX（东财版）** | >1.0 → 4 分；0.3~1.0 → 3 分；0~0.3 → 1 分；<0 → 0 分                                         | /4   |
| **今日增仓占比**  | >20% → 3 分；10~20% → 2 分；<10% → 1 分                                                      | /3   |
| **换手率**        | 8~20%（底部换手充分）→ 2 分；其他适中 → 1 分；>35%或<3% → 0 分                               | /2   |

**首板评分等级：**

- 18~22 分：🔴 **强** — 重点关注，次日可积极布局
- 13~17 分：🟠 **中** — 观望为主，等待明确信号

**连板评分等级：**

- 18~22 分：🔴 **强** — 连板龙头，重点关注
- 13~17 分：🟠 **中** — 可跟，但需控制仓位
- ≤12 分：🔵 **弱** — 回避，溢价过高或动能衰减

**强势股评分等级：**

- 14~18 分：🔴 **强** — 可重点关注，有涨停预期

## 第六步：龙头识别

**同一板块有多只涨停时，识别龙头股。** 龙头股综合评分额外 +2 分。

龙头判断条件（同时满足 3 条以上）：

1. 连板数最多
2. 封成比最高（封板最坚决）
3. 首次封板时间最早
4. 流通市值适中（30~150 亿，不是最大的）
5. DDX 全板块最高
6. 换手率适中（连板股低换手，首板股充分换手）

---

## 第七步：特别警示条件

触发以下任意条件，在该股旁标注 ⚠️，并说明原因：

**封板类警示：**

1. 尾盘偷袭：首封时间 > 14:00
2. 频繁炸板：炸板次数 ≥ 3
3. 封板资金过小：流通市值 > 50 亿 但封板资金 < 5000 万

**资金类警示：** 4. DDX 为负：主力净流出，不管涨幅多高都需警惕 5. 今强 3 日弱：今日增仓 > 15% 但 3 日增仓 < 8%（一日游风险） 6. DDX 与主力净比背离：方向相反 7. 涨停但增仓弱：涨幅 ≥9.9% 且今日增仓 < 10%

**换手类警示：** 8. 连板高换手：连板数 ≥2 且换手率 > 12% 9. 连板无量：连板数 ≥3 且换手率 < 1%（可能是一字连板，流动性差）

## 第八步：次日操作参考

### 可以考虑介入的信号（满足 2 条以上）

| 信号             | 具体标准                                  | 说明         |
| ---------------- | ----------------------------------------- | ------------ |
| 竞价强势         | 集合竞价封单量 > 昨日成交量 5%            | 主力护盘     |
| 低开企稳         | 高开 2%以内，回落后在昨日涨停价附近获支撑 | 洗盘确认     |
| 开盘量能持续     | 开盘首 5 分钟成交量 > 昨日同时段 1.5 倍   | 资金认可     |
| 板块联动         | 同板块其他股票同步走强                    | 板块行情延续 |
| 缩量回调放量突破 | 小幅回调（≤3%）后再度放量上攻             | 主力控盘洗筹 |

### 应当放弃介入的信号（触发任意 1 条即放弃）

| 信号         | 具体标准             | 说明         |
| ------------ | -------------------- | ------------ |
| 高开大幅跳空 | 开盘高开 ≥5%         | 获利盘压力大 |
| 开盘即砸盘   | 竞价后立刻大卖单压盘 | 主力出货     |
| 板块整体崩塌 | 同板块多只低开或跌停 | 逻辑崩塌     |
| 大盘急跌     | 指数开盘跌幅 >1.5%   | 系统性风险   |
| 竞价低开     | 次日竞价低开 ≥2%     | 人气散了     |

### 止损设置

- **激进止损**：跌破昨日涨停价立刻离场（约 -9%）
- **稳健止损**：跌破昨日收盘价 3% 以上离场
- **时间止损**：开盘 30 分钟内无法企稳，不管价位直接离场

## 数据缺失处理

| 缺失字段     | 处理方式                                                          |
| ------------ | ----------------------------------------------------------------- |
| 3 日增仓占比 | 跳过持续性分析，注明"仅单日数据"                                  |
| DDX          | 主力分析仅依赖主力净比，降低权重；DDX 维度得 0 分但不影响其他维度 |
| 量涨速       | 跳过，强势股总分上限降为 16 分                                    |
| 主力净比     | 仅依赖 DDX                                                        |
| 连板数       | 默认首板，连板维度得最低分                                        |
| 封板资金     | 跳过封成比计算，仅用时间+炸板判断                                 |
| 大盘情绪数据 | 注明"未获取大盘情绪"，正常分析但降低操作建议可信度                |

## 注意事项

- **DDX 版本区分**：东方财富 DDX 阈值（>1.0 为强）≠ 通达信 DDX 阈值（>2.0 为强），不可混用
- **不预测涨跌**：只评价资金指标强弱和买点条件，禁止说"明天会涨"
- **操作建议用词**：只用"可关注/观望/回避"，禁止用"买入/建仓/加仓"
- **次日买点是条件，不是预测**：必须说"如果出现 X 信号，可以考虑介入"
- **止损必须提**：每个有操作建议的股票都要附上止损位
- **盘中数据注明时效**：盘中截图需注明"数据截至 XX 时"
- **首板/连板/强势股分开排名**：不用同一评分体系混排
- **板块联动优先提醒**：同板块多只强势股时，先点明板块逻辑再分析个股
